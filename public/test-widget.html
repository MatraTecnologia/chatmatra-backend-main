Widget MatraChat — Página de Teste

1. Configurar e Carregar Widget

Preencha os dados abaixo para testar o widget. O API Key é a chave do seu canal API criado no painel de canais. O botão flutuante aparecerá no canto inferior direito desta página.

Encontre em Canais → seu canal API → ícone de engrenagem → aba "Instalar"

2. Snippet de Instalação

Após preencher as configurações e clicar em "Carregar Widget", o snippet aparecerá aqui para você copiar e colar no seu site.

(preencha os campos acima e clique em "Carregar Widget")

3. Fluxo de Teste

  1. Clique no botão flutuante no canto inferior direito desta página
  2. Preencha o formulário de nome e e-mail e clique em "Iniciar conversa"
  3. Digite uma mensagem e envie
  4. No painel admin (localhost:3000), vá em Conversas — o contato deve aparecer
  5. Clique no contato e responda a mensagem pelo painel admin
  6. A resposta deve aparecer em tempo real no widget desta página (via SSE)
  7. Recarregue esta página — o widget deve pular o formulário e mostrar o histórico
